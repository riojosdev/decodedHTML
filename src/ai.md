# AI USAGE

None of the contents (images, source code or text) on this website are generated by an AI. I do my own "content sharking" and dedicate my personal time to learn & curate the content for you. I have even written an article on my tech blog about [how to deal with content sharks](/content-creation/dealing-with-content-sharks.html).

I don't like to make an AI do the dirty work for me. I mean, am I really that useless and uncreative to let an AI take away the only skill I possess. But I do try to add a reference back to the OP. If not, I might have privately requested the OP if I could be granted the permission to use their creative work on my own platforms.

I'll be updating this page when/if I use AI generated content. Stay Creative my fellow Hoomans!
